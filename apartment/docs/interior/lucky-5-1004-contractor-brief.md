# 구리 럭키아파트 5동 1004호 인테리어 업체 상담 브리프

- 기준일: 2026-05-13
- 목적: 업체 연락/실측 요청 시 동일한 범위로 견적을 받기 위한 요약

## 기본 정보

- 구축 20평대 아파트
- 스타일: 화이트/베이지 + 과하지 않은 우드 포인트
- 예산 기준: 기본 3천만원 전후에서 시작하되, 거실 확장/샷시/제작가구 등 옵션은 분리 견적 요청
- 시스템에어컨: 설치하지 않음

## 희망 공사 범위

- 도배
- 바닥: 거실/주방/복도/현관/확장부 600×600 타일 희망, 방은 마루/강마루 계열 검토
- 욕실 전체 공사 + 욕실 선반/수납 검토
- 싱크대
- 조명
- 문/몰딩
- 전기/콘센트/스위치
- 중문 설치
- 거실 베란다 확장 + 단열 강화
- 거실 외창 샷시 교체 검토
- 안방 일자형 붙박이장
- 작은방 스타일러 자리 확보 + 붙박이장/수납장 배치 검토
- 안방 앞/화장실 입구 작은 화장대 + 거울/조명 + 드라이기용 콘센트

## 제외/유지 방향

- 시스템에어컨 제외
- 전체 샷시 교체 제외, 거실 외창 중심으로 검토
- 안방 쪽 베란다 비확장
- 보일러는 약 6년 전 교체가 맞으면 유지, 분배기/배관/누수/난방 순환만 점검

1. 거실 확장 가능 여부와 관리사무소/행위허가/신고 필요 여부
2. 거실 확장 시 단열/결로 방지 시공 방식
3. 거실 외창 샷시만 교체해도 단열 성능이 충분한지
4. 전체 샷시 미교체 시 나머지 샷시 보수 필요 여부
5. 600각 타일 시공 시 바닥 수평/난방 배관/단차 리스크
6. 중문은 3연동 슬라이딩/원슬라이딩/여닫이 중 어떤 타입이 평면에 적합한지
7. 안방 앞/화장실 입구 화장대 설치 시 통행폭/문 간섭/습기/콘센트 가능 여부
8. 욕실 선반/수납을 젠다이/매립 선반/코너 선반/거울장 중 어떤 방식으로 넣는 것이 적합한지
9. 세탁실에 LG FX24ENE 세탁기 + RD21ESE 건조기 직렬 설치가 가능한지
10. 보일러/분배기/배관 상태 점검 가능 여부
11. 기존 에어컨 배관/실외기 위치 사용 가능 여부
12. 냉장고장 제작 시 LG J825MEE042 냉장고 치수와 급수 연결, 방열 여유, 문 열림 각도를 반영할 수 있는지
13. 싱크대 하부장에 12인용 빌트인 식기세척기 기준으로 설치 공간과 급수·배수·전원·걸레받이·도어 간섭을 반영할 수 있는지
14. 아일랜드 식탁/장에 밥솥·에어프라이어·전자레인지 배치와 로봇청소기 도킹 공간을 함께 반영할 수 있는지
15. 작은방 붙박이장/수납장 제작 시 스타일러 공간을 비워두고 콘센트/문열림/환기 여유를 확보할 수 있는지

## 가전 치수 메모

### 냉장고

- LG 디오스 오브제컬렉션 STEM 얼음정수 냉장고 J825MEE042
- 설계 기준 치수 메모: W914 × H1792 × D918mm
- 주의: 최종 냉장고장 제작 전 제품 라벨/설치 매뉴얼/실물 기준으로 재확인 필요
- 업체 확인 필요:
  - 좌우·상부 방열 여유
  - 문 열림 각도와 벽/장 간섭
  - 콘센트 위치
  - 얼음정수 기능용 급수 연결: 싱크대 급수 분기, 배관 노출/몰딩/타공, 유지보수 접근성
