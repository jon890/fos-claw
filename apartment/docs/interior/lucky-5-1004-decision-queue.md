# 구리 럭키아파트 5동 1004호 의사결정 클리어 큐

- 기준일: 2026-05-18
- 목적: 다음 주 인테리어 업체 방문 전까지 매일 아침 결정할 항목을 3개씩 고르고, 확정된 항목은 다시 질문으로 올리지 않기.
- 성격: 이 파일은 코드에 박힌 하드코딩 목록이 아니라, 매일 갱신되는 의사결정 상태판이다. 아래 `다음 남은 질문`은 초기 시드이며, 아침 작업 때 의사결정 노트/현장 체크리스트/업체 상담 브리프를 읽고 누락·완료 상태를 반영해 갱신해야 한다.
- 진행 방식:
  - 아침 추천/리포트에서는 `다음 남은 질문`을 우선 보되, 먼저 관련 문서에서 새 미결정/완료 항목을 반영해 큐를 갱신한다.
  - 그날 질문은 `진행 중/미결정/실측 필요` 항목 중 3개만 고른다.
  - 사용자가 답하면 해당 항목을 `클리어 완료`로 이동하거나 상태를 `결정/방향 결정/현장 확인 후 최종 결정`으로 갱신한다.
  - `클리어 완료` 항목은 새 근거가 생기지 않는 한 다음 아침 질문에 다시 등장하지 않는다.
  - 실측으로만 풀리는 항목은 “결정 질문”이 아니라 “업체 확인 체크”로만 남긴다.

## 아침 3개 질문 선택 규칙

1. 먼저 `lucky-5-1004-interior-decisions.md`, `lucky-5-1004-decision-summary.md`, `lucky-5-1004-field-checklist.md`, `lucky-5-1004-contractor-brief.md`를 훑어 새 미결정/완료 항목이 있으면 이 큐를 갱신한다.
2. 갱신된 `다음 남은 질문`에서 우선순위가 높은 항목 3개를 고른다.
3. 최근 7일 리포트에서 이미 질문한 주제는 제외한다.
4. 이미 `클리어 완료`, `결정`, `방향 결정`, `현장 확인 후 최종 결정`인 항목은 질문으로 올리지 않는다.
5. 질문은 A/B/C 선택지 형태로 짧게 만든다.
6. 답변을 받으면 `lucky-5-1004-interior-decisions.md`, `lucky-5-1004-decision-summary.md`, 필요 시 `field-checklist.md`와 `contractor-brief.md`에 반영한다.

## 클리어 완료

### Q-001. 시스템에어컨

- 상태: 결정
- 결정: 설치하지 않음
- 이유: 예산 부담과 공사 부담 대비 효용이 낮다고 판단
- 후속: 기존 에어컨 배관/실외기 위치 사용 가능 여부만 확인

### Q-002. 바닥재

- 상태: 방향 결정
- 결정: 방까지 포함해 전체 600×600 타일 선호
- 후속: 방까지 타일 적용 시 차가움/딱딱함/하자/매도 호불호 리스크를 확인하고, 바닥 수평·난방 배관·줄눈 오염·단차 처리를 견적서에 명시

### Q-003. 중문 설치/타입

- 상태: 결정
- 결정: 중문 설치, 타입은 슬라이드 계열로 결정. 여닫이 중문은 동선 간섭 우려로 제외.
- 후속: 3연동 슬라이드 vs 원슬라이딩/초슬림 슬라이드 중 실측 후 선택

### Q-004. 안방 쪽 베란다

- 상태: 결정
- 결정: 비확장 유지 + 마감 개선
- 범위: 바닥 타일 교체, 벽면 페인트, 끝쪽 수납장 제작 희망
- 후속: 기존 창/문 상태, 결로/곰팡이/누수 흔적, 바닥 구배/배수, 수납장 결로 간섭 확인

### Q-005. 안방 붙박이장

- 상태: 방향 결정
- 결정: 기본 일자형 붙박이장
- 이유: 안방이 아주 넓지는 않아 ㄱ자/L자 수납은 답답해질 가능성이 큼
- 후속: 침대 사이즈와 설치 벽면 길이 실측

### Q-006. 안방 앞/화장실 입구 화장대

- 상태: 진행 방향 결정
- 결정: 안방 앞/화장실 입구 화장대 설치 방향, 단 실측 확인 필요
- 후속: 통행폭, 문 간섭, 습기, 콘센트 신설 가능 여부 확인

### Q-007. 세탁기+건조기 직렬 설치

- 상태: 방향 결정, 업체 확인 필요
- 결정: 세탁실 직렬 설치 1순위. 불가하면 세탁기는 세탁실, 건조기만 베란다 배치 + 물통 방식 검토
- 주의: 베란다 관이 우수관이면 세탁/건조기 배수호스 연결은 하지 않는 방향
- 제품: LG FX24ENE 세탁기 + RD21ESE 건조기
- 치수 메모: 세탁기 약 W700 × H990 × D830mm, 건조기 약 W700 × H990 × D820mm 수준으로 보고 공식/업체 재확인
- 후속: 문폭, 천장고, 창문 간섭, 급수/배수/콘센트 위치, LG 설치 기준, 베란다 배수관 성격 확인

### Q-008. 보일러

- 상태: 현장 확인 후 최종 결정
- 방향: 6년 전 교체가 맞고 정상 작동이면 유지
- 후속: 명판상 제조년월, 온수/난방, 분배기/배관 누수 확인

### Q-009. 거실 외창 샷시

- 상태: 현장 확인 후 최종 결정
- 방향: 거실 확장 시 거실 외창 교체 유력, 전체 샷시는 제외
- 후속: 단창/이중창, 페어유리, 외풍/결로/누수 확인

### Q-010. 작은방 스타일러 + 붙박이장 배치

- 상태: 방향 결정, 실측 필요
- 결정: 작은방에 스타일러 자리를 확보하고, 남는 벽면에 붙박이장/수납장을 배치한다.
- 임시 기준: 스타일러 모델 미확정이므로 W700 × D700 × H2000mm 내외 여유공간을 남기는 방향.
- 후속: 실제 스타일러 모델 치수, 문 열림 방향, 콘센트 위치, 환기/방열 여유, 붙박이장 필러/마감판 간섭 확인
