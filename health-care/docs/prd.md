# PRD — health-care

## 목적

개인 건강 기록, 재활 경과, 진료 준비를 장기적으로 관리한다. 의료진 판단을 대체하지 않고, 안전한 기록·요약·체크인 자동화를 제공한다.

## 현재 MVP

슬개골 불안정감/재활 트랙을 대상으로 3개 skill 경계를 설계한다.

| 기능 | 목적 | 자동 판단 범위 |
|---|---|---|
| daily-knee-rehab-checkin | 매일 아침 보수적 운동/주의사항 체크인 | 저장된 단계와 일반 플랜 기반 안내만 |
| knee-progress-intake | 사용자가 남긴 증상/반응을 구조화해 누적 | 사용자가 말한 사실 정리만 |
| weekly-knee-clinic-summary | 주간 경과 및 병원 제출 요약 초안 | Claude 기반 요약, 진단/처방 금지 |

## 성공 기준

- 민감한 원본 의료정보는 `data/`에만 남고 git 커밋되지 않는다.
- 공개 가능한 정책과 일반 플랜은 `config/`에서 관리한다.
- 매일 체크인은 짧고 보수적이며 중단 기준을 항상 포함한다.
- 주간 요약은 확정 사실, 사용자 보고, OCR 불확실성, 확인 필요를 구분한다.

## 비범위

- 진단, 처방, 약물 변경 지시
- 의료기관/의료진 판단 대체
- 공개 블로그 발행
- 플랫폼 내부 식별자 저장/커밋
